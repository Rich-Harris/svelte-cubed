# svelte-cubed

A Three.js component library for Svelte.

https://svelte-cubed.vercel.app

## Developing

This repo uses [pnpm](https://pnpm.io/). Install it...

```
npm install -g pnpm
```

...then install dependencies:

```
pnpm install
```

Run `pnpm dev -- --open` to launch the demo site.
